#### ID: TFNFR33 - Category: Code Style - Precise Local Types

Precise local types **SHOULD** be used.

Good example:

```terraform
{
  name = "John"
  age  = 52
}
```

Bad example:

```terraform
{
  name = "John"
  age  = "52" # age should be number
}
```
